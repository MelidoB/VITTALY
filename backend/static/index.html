Vittaly API Documentation

Click the arrows below to view instructions for each API endpoint.

Users API

POST /api/users/
Create a new user. Send a JSON payload containing:
{ "user_id": "string", "username": "string", "email": "valid email", "password_hash": "string" }

GET /api/users/
Retrieves a list of all users.

GET /api/users/{user_id}
Retrieves a user by user ID.

PUT /api/users/{user_id}
Updates a user (send full user JSON payload).

DELETE /api/users/{user_id}
Deletes a user by user ID.

Meals API

POST /api/meals/
Create a new meal. JSON payload should include:
{ "meal_id": "string", "user_id": "string", "meal_name": "string", "entry_type": "manual or scan" }

GET /api/meals/
Retrieves a list of meals.

GET /api/meals/{meal_id}
Retrieves a meal by meal ID.

PUT /api/meals/{meal_id}
Updates a meal with a full JSON payload.

DELETE /api/meals/{meal_id}
Deletes a meal by meal ID.

Nutrition API

POST /api/nutrition/
Create nutrition info for a meal. Example JSON:
{ "nutrition_id": "string", "meal_id": "string", "calories": number, "protein": number, "carbs": number, "fats": number, "fiber": number }

GET /api/nutrition/
Retrieves nutrition info for meals.

GET /api/nutrition/{nutrition_id}
Retrieves a nutrition record by ID.

PUT /api/nutrition/{nutrition_id}
Updates a nutrition record.

DELETE /api/nutrition/{nutrition_id}
Deletes a nutrition record.

Goals API

POST /api/goals/
Create a new goal. Example JSON:
{ "goal_id": "string", "user_id": "string", "daily_calories": number }

GET /api/goals/
Retrieves all goals.

GET /api/goals/{goal_id}
Retrieves a goal by ID.

PUT /api/goals/{goal_id}
Updates a goal record.

DELETE /api/goals/{goal_id}
Deletes a goal record.

Leaderboard API

POST /api/leaderboard/
Create a leaderboard entry. Example JSON:
{ "leaderboard_id": "string", "user_id": "string", "total_score": number }

GET /api/leaderboard/
Retrieves leaderboard entries.

GET /api/leaderboard/{leaderboard_id}
Retrieves a leaderboard entry by ID.

PUT /api/leaderboard/{leaderboard_id}
Updates a leaderboard entry.

DELETE /api/leaderboard/{leaderboard_id}
Deletes a leaderboard entry.
